## What Is the Artapli Embroidery Lettering Tool?

The Artapli Embroidery Lettering Tool is an online web application that allows embroidery enthusiasts to:

- Type any word or phrase
- Visually arrange individual embroidery letters on a true-to-scale virtual hoop canvas
- Export the finished design as a ready-to-stitch machine embroidery file

### Supported File Formats

The tool exports 8 industry-standard embroidery file formats:
- **PES** - Brother/Babylock
- **DST** - Universal format
- **JEF** - Janome
- **VP3** - Husqvarna Viking
- **EXP** - Melco/Bernina
- **HUS** - Husqvarna Viking (older)
- **XXX** - Singer
- **VIP** - Pfaff

### Platform Requirements

- **Platform**: Web browser (no installation required)
- **Devices**: Windows PC, Mac, Chromebook, iPad, Android tablet, smartphones
- **Browsers**: Chrome, Safari, Firefox, Edge
- **Cost**: Free to use with purchased Artapli embroidery fonts
- **Internet**: Required for use

## How It Works

Visit [artapli.shop/pages/lettering-app](https://artapli.shop/pages/lettering-app) in any modern browser
