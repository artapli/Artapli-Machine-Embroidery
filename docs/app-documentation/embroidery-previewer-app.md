## Overview

Embroidery Previewer is free web-based visualization tool by Artapli. Displays realistic preview of embroidery designs without requiring embroidery software.

## Purpose

- View designs before purchasing
- Preview designs before stitching
- Check design appearance without software
- Share visual previews with clients
- Verify design details

## Access

- **URL**: [artapli.shop/pages/embroidery-previewer-app](https://artapli.shop/pages/embroidery-previewer-app)
- **Platform**: Web browser (all devices)
- **Cost**: Free
- **No login required**

## Key Features

1. Realistic stitch rendering
2. Multiple view modes (3D, realistic, schematic)
3. Color customization
4. Zoom and pan
5. Rotate design
6. Thread brand selection
7. Background color options
8. Screenshot/export preview

## Supported File Formats

Previewer reads:

- DST, PES, JEF, HUS, EXP, VIP, XXX
- 50+ embroidery formats
- Most common formats supported

## How to Use Embroidery Previewer

### Step 1: Load Design

- Visit Embroidery Previewer page
- Click "Upload Design" or drag-and-drop
- Select embroidery file
- Preview generates automatically

### Step 2: Choose View Mode

**Options**:

- **Realistic View**: Shows design as it will stitch
- **3D View**: Shows texture and dimension
- **Schematic View**: Technical view with stitch paths
- **Outline View**: Simplified shape view

### Step 3: Customize Display

- Change background color (white, black, fabric textures)
- Adjust zoom level
- Rotate design
- Pan to view different areas

### Step 4: Customize Colors (Optional)

- Click color in palette
- Choose new thread color
- Select thread brand (Madeira, Isacord, etc.)
- Preview updates instantly

### Step 5: Export or Screenshot

- Use browser screenshot
- Share preview image
- Save for reference

## View Modes Explained

### Realistic View

- Most accurate representation
- Shows stitches as they'll appear on fabric
- Best for final verification
- Simulates actual thread sheen

### 3D View

- Shows design dimension and texture
- Highlights raised stitches (satin, column)
- Useful for assessing depth
- Great for presentation

### Schematic View

- Technical view showing stitch paths
- Displays stitch direction
- Shows underlay stitches
- Useful for technical analysis

### Outline View

- Simplified shape display
- Fast rendering
- Good for size verification
- Shows basic structure

## Limitations

### Previewer does NOT:

- Edit designs
- Convert formats
- Modify files
- Save designs with new colors (preview only)
- Show fabric pucker or tension issues
- Simulate specialty threads (metallic, glow-in-dark)

For editing, use embroidery software.

## Troubleshooting

### Issue: Design won't preview

**Solutions**:
- Check file format supported
- Verify file not corrupted
- Try different browser
- Check file size < 10MB

### Issue: Colors look wrong

**Solutions**:
- Adjust thread brand selection
- Check monitor color calibration
- Try different background color
- Remember: Screen colors differ from actual thread

### Issue: Preview is slow

**Solutions**:
- Reduce design complexity if possible
- Close other browser tabs
- Try different device
- Wait for processing (very complex designs take time)

### Issue: Can't see details

**Solutions**:
- Use zoom feature
- Try different view mode
- Adjust background color for contrast
