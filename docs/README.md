## Key Topics

### Font Formats
- **ESA** - Wilcom, Hatch, Bernina ART
- **BX** - Embrilliance (NOT compatible with Wilcom)
- **PES Alphabet** - PE-Design
- **EOF** - Embird

### File Formats
- **DST** - Universal format, works on all machines
- **PES** - Brother/Babylock machines
- **JEF** - Janome machines
- **EXP** - Bernina/Melco machines
- **HUS** - Husqvarna machines
- **VIP** - Pfaff machines

Learn about handling:
- Fringe/Fluffy designs

## Important Notes

### Critical Information for All Users

1. **BX Fonts**: BX fonts do NOT work with Wilcom. BX fonts are designed exclusively for Embrilliance software.

2. **Specialty Designs**: Never resize or convert fringe, puffy foam, or chain stitch designs. Use the exact format and size provided.

3. **File Conversion**: Always disable "Optimize stitches" and similar automatic optimization features when converting formats.

4. **Hatch 3 Users**: You can map purchased alphabets to KDC (keyboard font). Watch the tutorial: https://www.youtube.com/watch?v=2nVwUMMbE-Y
