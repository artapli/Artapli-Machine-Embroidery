## What is ESA

ESA (Embroidery Sequence Archive) is Wilcom's proprietary alphabet format. It stores embroidery letters as a mapped font, enabling typing instead of placing individual letters.

## ESA File Structure

ESA files contain:

- Individual letter/character designs (A-Z, a-z, 0-9, punctuation)
- Left bearing (space before letter)
- Right bearing (space after letter)
- Baseline position (vertical alignment)
- Default sizing information
- Kerning pairs (spacing between specific letter combinations)

## Benefits of ESA Format

Compared to individual letter files:

- Type continuous text instead of placing letters
- Automatic letter spacing
- Consistent baseline alignment
- Easy editing (change text without repositioning)
- Professional results
- Time savings (minutes vs hours for long text)

## Technical Considerations

ESA files are binary format (not human-readable):

- Cannot be edited in text editor
- Require Wilcom-family software to modify
- Size varies (typically 50KB - 2MB depending on complexity)
- Cross-platform (Mac and Windows with compatible software)

## ESA Font Properties

After importing, you can modify in Font Manager:

- **Default sizes** (small, medium, large presets)
- **Letter spacing** (percentage adjustment)
- **Baseline shift** (for special effects)
- **Envelope settings** (curved or arced text)

## Creating ESA from Individual Letters (Advanced)

Wilcom and Hatch allow creating ESA from letter files:

1. Import all letters (A-Z, 0-9, punctuation)
2. Font Manager > Create New Alphabet
3. Assign each letter to keyboard key
4. Set spacing (measure and input bearing values)
5. Define baseline
6. Test and refine spacing
7. Save as ESA file

This requires digitizing knowledge. Artapli provides ESA when creating fonts, saving customers this complex process.

## ESA Version Compatibility

ESA format has versions:

- **Older ESA** (Wilcom E2, E3 era)
- **Modern ESA** (Wilcom E4, Hatch 2+)

Artapli ESA fonts are created in modern format. If you have very old Wilcom:

- Update software (recommended)
- Or contact Artapli for legacy format

## Troubleshooting ESA

### Issue: ESA won't import

**Solutions**:
- Verify software version (Wilcom/Hatch/Bernina ART)
- Update software to latest version
- Check file is not corrupted (re-download)
- Ensure .ESA extension is present
- Try manual import via Font Manager

### Issue: Font appears but letters overlap

**Solutions**:
- Adjust letter spacing in text properties
- Check baseline settings
- Some fonts designed with tight spacing (intentional for specific looks)

### Issue: Missing characters

**Solutions**:
- Check product description (some fonts are uppercase only)
- Verify complete ESA installed (not partial)
- Some specialty fonts include limited character sets

### Issue: Sizing problems

**Solutions**:
- Use size presets built into ESA
- Avoid custom sizing on specialty fonts (fringe, puffy)
- For standard fonts, 10-15% resize is safe
