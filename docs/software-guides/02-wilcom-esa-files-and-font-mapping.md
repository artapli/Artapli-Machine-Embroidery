## What is ESA Format

ESA is Wilcom's proprietary alphabet format that allows fonts to be mapped and typed like regular keyboard text instead of placing individual letters.

## Benefits of ESA Fonts

1. Type words continuously instead of placing individual letters
2. Automatic letter spacing and kerning
3. Baseline alignment handled automatically
4. Much faster for creating text designs
5. Easy to edit and modify text after creation

## How to Install ESA Fonts in Wilcom

### Step 1: Locate ESA Files

- Download Artapli font package
- Extract ZIP file
- Find files with .ESA extension
- Note: Not all fonts include ESA format (check product description)

### Step 2: Install to Wilcom

**Method A (Automatic)**:
- Double-click the ESA file
- Wilcom automatically imports and installs
- Font appears in Lettering toolbar

**Method B (Manual)**:
- Open Wilcom
- Go to Edit > Lettering > Font Manager
- Click "Add Font"
- Navigate to ESA file location
- Select ESA file and click Open
- Font is now available in font list

### Step 3: Using Mapped Fonts

- Select Text tool (Lettering toolbar)
- Choose your installed Artapli font from dropdown
- Type your text
- Font appears with proper spacing and baseline
- Adjust size, spacing as needed

## Font Mapping Technical Details

ESA files contain:

- Individual letter designs for A-Z, a-z, 0-9
- Punctuation marks
- Spacing data (left/right bearing)
- Baseline position
- Default sizing information
- Kerning pairs for common letter combinations

## Artapli Fonts with ESA Format

Most Artapli alphabets include ESA format. Check product description for:
"ESA format included for Wilcom font mapping"

If ESA not included, you can:

1. Use individual letter files
2. Create your own font mapping (advanced)
3. Request custom ESA format (contact support)

## Troubleshooting ESA Fonts

### Issue: ESA file won't import

**Solution**: Check Wilcom version compatibility. Older Wilcom versions may not support newer ESA formats. Contact Artapli for compatible version.

### Issue: Letters overlap or spacing is wrong

**Solution**: Use Lettering > Spacing adjustments. ESA fonts include default spacing but can be customized per project.

### Issue: Missing characters

**Solution**: Check if all letters are included. Some specialty fonts only include uppercase or specific character sets.

## Creating Text with ESA Fonts

1. Select Lettering tool
2. Choose Artapli font from list
3. Type text in text box
4. Adjust:
   - **Size**: Use size slider or enter specific height
   - **Spacing**: Adjust letter spacing percentage
   - **Baseline**: Shift for special effects
   - **Envelope**: Apply curves or arcs to text

## Advanced ESA Features

- **Monogramming**: Combine letters with special positioning
- **Circle lettering**: Apply text to curved paths
- **Multiple lines**: Create paragraphs with proper line spacing
- **Color changes**: Assign different colors per letter or word

## Compatibility

ESA format is Wilcom-specific. Other software equivalents:

- **Hatch**: Also uses ESA format
- **Embrilliance**: Uses BX format (see Embrilliance section)
- **PE-Design**: Uses PES alphabet format
- **Embird**: Uses font objects
